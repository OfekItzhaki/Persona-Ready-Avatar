# Persona-Ready-Avatar
A high-fidelity "Digital Human" frontend that renders synchronized talking avatars driven by AI-generated speech.

Key Features:

Real-time Lip Sync: Dynamic 3D mouth animation driven by Azure Neural TTS viseme data.

Avatar Engine: Integrated Three.js/React-Three-Fiber support for customizable GLB/Ready Player Me models.

Multilingual Voice: Automated voice selection based on the brain's response language.

Decoupled Architecture: Purely visual/auditory client that consumes any LLM/RAG backend via a standardized API.
